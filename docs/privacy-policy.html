Privacy Policy

English Learning Tracker — Last updated: March 26, 2026

Overview

English Learning Tracker is a Chrome extension that helps users build daily English learning habits. We are committed to protecting your privacy. This policy explains what data the extension handles and how it is used.

Data Storage

All user data is stored locally on your device using the Chrome Storage API. This includes:

  • Daily practice records and streak data
  • Saved vocabulary entries
  • Writing entries and AI feedback
  • Cached news articles
  • Your settings and preferences

No user data is sent to or stored on any server controlled by us.

Third-Party API Services

The extension sends data to third-party APIs that you configure with your own API keys:

  • New York Times API — to fetch news articles for reading practice
  • AI providers (OpenAI, Anthropic, Moonshot/Kimi, DeepSeek, Google Gemini) — to generate quiz questions, review writing, and explain vocabulary. The text you write or select is sent to the AI provider you choose.
  • Text-to-Speech providers (OpenAI TTS, ByteDance OpenSpeech/Volcengine) — to generate audio for listening practice. AI-generated passage text is sent to the TTS provider you configure.

Your API keys are stored locally and are only sent directly to their respective services. We never see or collect your API keys.

Data Collection

This extension does not:

  • Collect any personal information
  • Track your browsing activity
  • Use analytics or tracking services
  • Send data to any server we operate
  • Share data with any third party (other than the API services you configure)

Permissions

The extension requests the following permissions, used solely for the described purposes:

  • Storage / Unlimited Storage — to save your learning data locally
  • Host permissions — to call AI and TTS APIs, and to inject word explanation popups
  • Active Tab / Scripting — to show word explanations on the current page when you right-click a word
  • Context Menus — to add the "Explain" right-click menu option
  • Alarms — for daily streak reset at midnight
  • Side Panel — to show a quick task overview panel

Data Export and Deletion

You can export all your data as a JSON file from the Settings page. To delete all data, uninstall the extension or clear its storage from Chrome's extension settings.

Changes to This Policy

If we update this privacy policy, the changes will be posted on this page with an updated date.
