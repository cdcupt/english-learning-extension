## Detailed Description

English Learning Tracker helps you build consistent English learning habits through 5 daily practice tasks.

**Reading**
Read curated New York Times articles and test your comprehension with AI-generated quizzes covering main ideas, vocabulary, and inference.

**Writing**
Practice writing on random topics and get detailed AI feedback — grammar corrections, style suggestions, and a score from 0-100.

**Vocabulary**
Build your word list by right-clicking any word on any webpage to get an AI-powered explanation. Review words with flashcard quizzes.

**Listening**
IELTS-style listening practice with AI-generated passages and HD audio. Choose from OpenAI TTS or ByteDance OpenSpeech for natural-sounding speech. Answer 5 comprehension questions after each listen.

**Speaking**
Track your daily speaking practice with a simple check-in.

**More Features**
• Streak tracking — complete all 5 tasks for a perfect day
• Weekly summary with completion stats
• Practice history by date
• Side panel for quick task status
• Multiple AI providers: Kimi, OpenAI, Claude, DeepSeek, Gemini
• Configurable daily targets (1-5 articles, 1-5 listening practices)
• Export all data as JSON

**Setup**
You'll need API keys to get started:
1. NYT API key (free at developer.nytimes.com)
2. An AI provider API key (Kimi, OpenAI, Claude, DeepSeek, or Gemini)
3. Optional: TTS API key for HD listening audio (OpenAI or ByteDance/Volcengine)
